# Multipurpose Robotic Sanitizing Machine

## Description
This project focuses on the design and development of a semi-autonomous robotic sanitizing machine for public areas such as schools, hospitals, theaters, and bus stands.

## Technologies Used
- Arduino
- Embedded C
- Python
- Ultrasonic Sensors
- DC Motors

## Features
- Autonomous movement
- Obstacle detection using ultrasonic sensors
- Automated sanitization mechanism
- Low-cost and scalable design

## Outcome
Reduced manual sanitization effort by approximately 60%.
